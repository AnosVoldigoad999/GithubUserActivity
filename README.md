# GITHUBUSERACTIVITY

GitHub User Activity
Sample solution for the [GitHub User Activity](https://roadmap.sh/projects/github-user-activity) challenge from [roadmap.sh](roadmap.sh).

How to run
Clone the repository and run the following command:

git clone https://github.com/AnosVoldigoad999/GithubUserActivity.git
cd GithubUserActivity
Run the following commands to install dependencies, build, install and run the project:
#
NOTE: YOU NEED TO HAVE GO INSTALLED, IF YOU DON'T HAVE IT INSTALLED GO HERE: https://go.dev/doc/install
#
go mod download
#
go build
#
go install
#
gat --help # To see the list of available commands

# To get recent activity
gat username
#
e.g gat AnosVoldigoad999
